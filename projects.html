CloudTik - ISP Management System

Overview

CloudTik is a web-based ISP management system designed to handle customer subscriptions, billing, and network access across multiple camps. The system focuses on scalability, automation, and real-time network control.

Screenshots

Dashboard overview with real-time statistics

Tech Stack
  • Laravel (Backend)
  • Blade (Frontend)
  • MySQL (Database)
  • MikroTik RouterOS API (PHP)
Key Features
  • Subscription Management System
    • Customizable packages (e.g., 15-day, 30-day plans)
    • Flexible pricing configuration
    • Subscription lifecycle tracking with statuses(Active, Running, Expired, Canceled, Transferred)
  • Customer Management
    • Customer registration required for subscriptions
    • Recovery support for lost subscriptions
    • Real-time tracking of login and expiry timestamps
  • Multi-Camp Management
    • Manage multiple camps within a single system
    • Transfer customers between camps seamlessly
  • Access Control System
    • Role-based page access control using permission toggles
    • Supports multiple salespersons operating simultaneously in the same camp
  • Network Integration (MikroTik)
    • Integrated with MikroTik using RouterOS API
    • Automated hotspot user creation and removal
    • Real-time session handling via API
  • Automation
    • Cron jobs to automatically handle expired subscriptions
    • Ensures system consistency without manual intervention
My Role & Contributions
  • Designed and developed the full system architecture
  • Built backend APIs and business logic using Laravel
  • Integrated MikroTik routers via RouterOS API for network control
  • Implemented subscription lifecycle management system
  • Developed multi-camp and role-based access control features
  • Created responsive UI using Blade templates

OneCode - Advanced ISP Management System

Overview

OneCode is an advanced ISP management system that extends the capabilities of CloudTik by introducing a flexible access model with both subscriptions and voucher-based authentication. The system is designed to support both long-term users and short-term visitors within labor camp environments.

Screenshots

Dashboard overview with real-time statistics

Tech Stack
  • Laravel (Backend)
  • Blade (Frontend)
  • MySQL (Database)
  • MikroTik RouterOS API
  • JavaScript (QR Code Generation)
Key Features
  • Dual Access System (Subscriptions + Vouchers)
    • Supports registered users with subscriptions
    • Enables temporary access via voucher codes for short-term users
    • Provides flexibility based on customer needs
  • Voucher-Based Authentication
    • Generates secure 8-digit numeric voucher codes
    • Voucher activation binds user’s MAC address via MikroTik
    • Prevents unauthorized sharing of access
    • Voucher validity period of up to 3 months
  • Polymorphic Access Plans
    • Implemented polymorphic relationships to handle both subscriptions and vouchers
    • Unified system design for managing different access types
  • QR Code Integration
    • QR code generated on MikroTik hotspot login page
    • Salesperson scans QR for faster customer handling
    • Enables quick device reset and MAC address updates
    • Improves speed and efficiency of the sales process
  • Enhanced User Experience
    • Faster onboarding for customers without registration
    • Transparent and user-friendly workflow
    • Improves customer trust and service accessibility
  • Core System Features
    • Includes all functionalities of CloudTik:
      • Subscription management
      • Customer handling
      • Multi-camp management
      • Access control and automation
My Role & Contributions
  • Designed and developed the full system architecture
  • Implemented voucher-based authentication system
  • Designed polymorphic database relationships for flexible access handling
  • Integrated MikroTik RouterOS API for network-level control
  • Developed QR-based workflow to optimize sales and support processes
  • Enhanced system usability for both salespersons and end users

Offline POS System

Overview

Developed a fully offline Point of Sale (POS) system to support retail businesses in managing sales, inventory, and daily operations without relying on internet connectivity.
The system was designed to be fast, reliable, and easy to use, making it suitable for small and medium-sized businesses. It was successfully deployed to 20+ customers, generating revenue for the company.

Screenshots

Dashboard overview with real-time statistics

Tech Stack
  • Backend: PHP (OOP)
  • Architecture: Custom MVC Pattern
  • Frontend: HTML, CSS, JavaScript, jQuery
  • Asynchronous Handling: AJAX
  • Database: MySQL (Managed via MySQL Workbench)
Key Features
  • Item & Category Management
    Create and manage item categories and product listings
  • GRN (Goods Received Note)
    Track incoming stock and update inventory accurately
  • Inventory Management
    Real-time tracking of stock levels and product availability
  • Billing & Invoicing
    Generate invoices with support for:
    • Percentage-based discounts
    • Custom discount values
  • Receipt Printing
    Printable receipts for completed transactions
  • Item Return Handling
    Process returns and adjust inventory accordingly
  • Reporting System
    Generate reports for sales, inventory, and business insights
  • Offline Functionality
    Fully operational without internet connectivity
