Projects

CloudTik - ISP Management System
Overview

CloudTik is a web-based ISP management system designed to handle customer subscriptions, billing, and network access across multiple camps. The system focuses on scalability, automation, and real-time network control.

Screenshots

Dashboard overview with real-time statistics

Tech Stack
  • Laravel (Backend)
  • Blade (Frontend)
  • MySQL (Database)
  • MikroTik RouterOS API (PHP)
Key Features
  • Subscription Management System
    • Customizable packages (e.g., 15-day, 30-day plans)
    • Flexible pricing configuration
    • Subscription lifecycle tracking with statuses(Active, Running, Expired, Canceled, Transferred)
  • Customer Management
    • Customer registration required for subscriptions
    • Recovery support for lost subscriptions
    • Real-time tracking of login and expiry timestamps
  • Multi-Camp Management
    • Manage multiple camps within a single system
    • Transfer customers between camps seamlessly
  • Access Control System
    • Role-based page access control using permission toggles
    • Supports multiple salespersons operating simultaneously in the same camp
  • Network Integration (MikroTik)
    • Integrated with MikroTik using RouterOS API
    • Automated hotspot user creation and removal
    • Real-time session handling via API
  • Automation
    • Cron jobs to automatically handle expired subscriptions
    • Ensures system consistency without manual intervention
My Role & Contributions
  • Designed and developed the full system architecture
  • Built backend APIs and business logic using Laravel
  • Integrated MikroTik routers via RouterOS API for network control
  • Implemented subscription lifecycle management system
  • Developed multi-camp and role-based access control features
  • Created responsive UI using Blade templates

Trizent Infratech – Network Solutions Company

Role: Full Stack Developer
Duration: from March 2025 to present

Overview

Trizent Infratech provides network infrastructure, CCTV installation, and internet services for labor camps. The company relied on manual spreadsheet-based workflows for managing customer subscriptions and sales operations.

Responsibilities & Contributions
  • Designed and developed the CloudTik ISP Management System to replace manual spreadsheet-based operations
  • Built a full-stack web application using Laravel, Blade, and MySQL
  • Integrated MikroTik routers using RouterOS API for automated user and session management
  • Developed features for customer registration, subscription handling, and real-time access control
  • Created multi-camp management functionality to support scalable operations
  • Implemented role-based access control for multiple salespersons
Challenges in Existing System
  • Sales and customer data managed manually using spreadsheets
  • No visibility into customer login time or subscription expiry
  • Limited to a single 30-day package with no flexibility
  • No customer history tracking (only basic details recorded)
  • No support for camp transfers or subscription recovery
  • Voucher-based system caused permanent access loss if credentials were misplaced
  • Difficult for salespersons to operate efficiently using mobile devices
Impact & Improvements
  • Replaced manual workflows with a centralized ISP management system
  • Enabled real-time tracking of customer login and subscription expiry
  • Introduced flexible subscription packages beyond fixed plans
  • Improved sales efficiency with a mobile-friendly system
  • Enabled customer history tracking and better data management
  • Reduced operational errors and improved overall system reliability
  • Streamlined sales operations across multiple camps
